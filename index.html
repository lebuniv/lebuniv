Social Network Model Definitions

Social Network Model

A social network model is a set of actors and the interactions between them. It is represented as a graph, where actors are nodes and interactions are links. Actors can be connected by various types of ties and relations.

Definitions

Degree

Degree is the sum of connections (or sum of the weights of connections in case of weighted graphs) from or to an actor.

Closeness Centrality

Closeness centrality is the distance of one actor to all others in the network. It measures how easily an actor can reach all other actors.

Betweenness Centrality

Betweenness centrality is the number of shortest paths that pass through an actor. It represents how frequently an actor is between other actors' geodesic paths.

Eigenvector

Eigenvector measures the importance of an actor in the network. It takes into account both the actor's own connections and the connections of their neighboring actors.

Out-degree

Out-degree is the sum of connections from an actor to others in the network.

In-degree

In-degree is the sum of connections to an actor from others in the network.

HITS

HITS (Hyperlink-Induced Topic Search) is a metric used in website analytics. A hit refers to a request made to a website for a particular file to be downloaded from the server. When a user clicks on a web page, regardless of the number of files downloaded, it is called a visit.
